## Rotation Logic

- **14 unique pairings** cycle through the year
- Each day shows a **different combination** of 2 cultures
- No client-side logic needed â€” server handles everything
- Pairings repeat every 14 days

## Example Usage

### iOS (Swift)
```swift
func fetchAncientWisdom(lang: String = "en") async throws -> AncientWisdomResponse {
    let url = URL(string: "https://source.thequietframe.com/api/wisdom/cycle?lang=\(lang)")!
    let (data, _) = try await URLSession.shared.data(from: url)
    return try JSONDecoder().decode(AncientWisdomResponse.self, from: data)
}
```

### Android (Kotlin)
```kotlin
suspend fun fetchAncientWisdom(lang: String = "en"): AncientWisdomResponse {
    return httpClient.get("https://source.thequietframe.com/api/wisdom/cycle") {
        parameter("lang", lang)
    }.body()
}
```
