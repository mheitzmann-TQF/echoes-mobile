## Getting Started

### Option 1: Run Locally

```bash
cd mobile
npm install
npx expo start
```

Then scan the QR code with **Expo Go** app on your phone.

### Option 2: Expo Snack

Upload this folder to [snack.expo.dev](https://snack.expo.dev) for instant preview.

### Option 3: Build for App Store

```bash
npx expo build:ios
npx expo build:android
```

## Configuration

### Environment Variables

Copy `.env.example` to `.env` and set your backend URL:

```bash
cp .env.example .env
```

```env
# Development (local)
EXPO_PUBLIC_API_URL=http://localhost:5000

# Production (deployed)
EXPO_PUBLIC_API_URL=https://your-backend.replit.app
```

The API client automatically uses the `EXPO_PUBLIC_API_URL` environment variable.
